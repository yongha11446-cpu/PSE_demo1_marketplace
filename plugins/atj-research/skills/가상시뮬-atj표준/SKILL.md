# 가상시뮬 — ATJ 표준 골격 (대안1 검증판)

리그노셀룰로오스 바이오매스 → 에탄올 → 에틸렌 → 올리고머 → 수소화 → 제트유(SAF)의
**블록별 물질수지 shortcut**을 표준 조건으로 세운다. 2026-06 ATJ_재현에서 대안1로 검증됨
(jet 수율 21.1% ≈ Romero biojet 21% 정합). 아스펜 본모사 전 타당성·수렴 사전점검용.

## 핵심 원칙 (반드시 지킴)
- **추측 금지.** 모든 전환율/선택도엔 근거(논문번호+이름, 저자·연도·Table/쪽) 병기. 미확인은 `확인 필요`.
- 새 바이오매스면 **조성을 원문에서 다시 확인**(md 믿지 말 것). 조성이 수율을 좌우한다.
- 무거운 계산은 `scripts/`의 파이썬으로, 대화창엔 결과 표만. 단위 항상 명시.

## 표준 블록 체인 + 검증된 조건 (대안1, 원문 PDF 직독 확정)
| 블록 | 기준논문 | 화학량론 | 전환율/선택도 | 운전 |
|---|---|---|---|---|
| 전처리·가수분해·발효 | 01.NREL2011 Humbird | 당화+발효 | glucan→glucose 90.5%·xylan→xylose 90%·glucose→EtOH 95%·xylose→EtOH 85% (이론 0.511 g/g) | 전처리158/효소48/발효32°C |
| 탈수→에틸렌 | 46.Ouyang2009 La-HZSM-5 | C2H5OH→C2H4+H2O | 전환 98.5%·에틸렌선택 99.5%·DEE 0% (Table2) | 260°C·상압·LHSV1.1·EtOH50wt% |
| 올리고머화 | 32.Romero2021 | nC2H4→CnH2n(질량보존) | ★전환분포(RStoke): C4 0.32·C6 0.24·C8 0.18·C10~20 각0.043 → **제트컷 C8–16 ≈35%** (jet선택 97% 아님!) | 120°C·35bar |
| 수소화 | 32.Romero2021 | CnH2n+H2→CnH2n+2 | 전환 0.99·H2 1:1 mol (Table3) | 100°C·15bar |
| 분리·증류 | 32.Romero2021 | RadFrac 3탑(16/33/87단), 핵심성분 99% 회수 | **biojet 수율 ≈21% (에탄올 대비, 피드 아님)** | 분리존5.31%↓·가열34.75%↓·냉각30.32%↓ |

> ⚠️ 흔한 함정: Romero "biojet 21%"는 **에탄올 대비**다(건조피드 대비 아님). 올리고 제트선택을 97%로 잡으면 ~2.7배 과대평가됨. 반드시 탄소수 분포로 C8–16 컷을 계산할 것. 검증: jet/EtOH ≈ 21%, jet/건조피드 ≈ 7.8%.

## 절차
1. **조성 확보**: 대상 바이오매스 조성(cellulose/hemicellulose/lignin/ash, dry wt%)을 원문에서 확인.
   - Kenaf core 표준값(검증): Ng2013 BioResources Table1 → 정규화 cellulose 43.1·hemi 35.3·lignin 17.7·ash 3.9 wt%.
   - 못 구하면 `논문탐색가` 서브에이전트로 web 발췌(저자·Table·쪽 병기).
2. **계산 스크립트**: 프로젝트 `scripts/가상시뮬_전체.py`를 본보기로 작성·수정.
   기준 처리량(basis, 임의·스케일가능) → 블록 함수 체인(전처리발효→탈수→올리고머화→수소화) 실행.
   다른 대안(촉매/통합공정)을 비교하려면 `대안들` 딕셔너리에 블록 조건만 바꿔 추가.
3. **수렴·타당성 판정**: 전환율 ≤100%·음수 없음·수지 닫힘오차(shortcut은 0%) 확인.
   재순환은 미고려임을 명시(본모사에서 수율 상향 여지). jet수율/피드를 문헌 biojet 수율과 교차검증.
4. **엑셀 정리**: 입력 JSON 작성 → `python "${CLAUDE_PLUGIN_ROOT}/scripts/make_vsim_excel.py" <JSON> <xlsx>`.
   대안별 시트(블록|논문번호+이름|온도|압력|전환율|선택도|스트림|수렴|근거) + 요약 시트.
5. **기록**: 확정 조건·결과를 `/업데이트 <블록>`으로 연구노트(`연구노트.md`)에 기록(같은 항목 덮어쓰기·새 항목 추가, 글로벌 `~/.claude/연구_라이브러리/마스터_연구노트.md`에도 반영).

## 다른 대안(비교용) 검증된 조건
- 고선택도: 11.Wu2017 ZSM-5 탈수(220°C·98.5%·선택100%) + 22.NiAlKIT-6(96%·C8+56%)/23.Li2025 tandem(jet64wt%).
- 통합·강화: 32.Romero + 33.Villareal(올리고+수소화 RD 통합, 가열34.75%↓·냉각30.32%↓; 반응수치 확인필요).
