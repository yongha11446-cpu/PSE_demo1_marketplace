# PSE_demo1 — 아스펜 공정모사 연구 비서 플러그인

리그노셀룰로오스 → SAF(ATJ) 같은 화학공정 모사 연구를, 문헌탐색부터 노벨티 도출·가상 시뮬레이션·연구노트 기록까지 한 번에 돕는 Claude Code 플러그인.

## 들어 있는 것

| 종류 | 이름 | 설명 |
|---|---|---|
| 커맨드 | `/주제분석` | 논문탐색·가치평가·노벨티·연구방향 3대안·엑셀 정리 |
| 커맨드 | `/분석시작` | 수집 논문 전체 정독 → 한계점 분류 → 노벨티 점수화 |
| 커맨드 | `/가상시뮬레이션` | 대안별 블록 수지·수렴·물리타당성 검증 + 엑셀 |
| 커맨드 | `/업데이트` | 확정 값·결정을 연구노트(색인+카드)에 anti-stale 기록 |
| 스킬 | `가상시뮬-atj표준` | 검증된 ATJ 대안1 골격으로 빠르게 jet 수율 추정 |
| 에이전트 | `논문탐색가` `정독분석가` `노벨티설계가` | 대량 검색·정독·노벨티를 병렬로 분리 |
| 훅 | 갱신 리마인드 | 아스펜 스트림 붙여넣기 감지 → 기록 프로토콜 자동 안내 |
| 헬퍼 | `make_refs` `make_lit_excel` `make_novelty_excel` `make_vsim_excel` | 엑셀·refs 자동 생성 (Python) |

## 요구사항

- **OS**: Windows (훅이 PowerShell 기반). Mac/Linux는 훅 일부가 동작하지 않음.
- **Python** + `openpyxl` — 엑셀 헬퍼용. 설치:
  ```
  pip install openpyxl
  ```

## 설치

```
/plugin marketplace add yongha11446-cpu/PSE_demo1_marketplace
/plugin install PSE_demo1@PSE_demo1
```

## 처음 한 번: 템플릿 복사

- **글로벌 연구노트**(프로젝트 횡단 색인): `/업데이트`·`/가상시뮬레이션` 첫 실행 시 자동 생성된다.
  미리 만들려면 `templates/마스터_연구노트.md`를 `~/.claude/연구_라이브러리/` 로 복사한다.
- **새 연구 프로젝트 규칙**: `templates/CLAUDE.md`를 프로젝트 **루트**로 복사해 `<...>` 부분을 채운다.
  (한국어·표·단위 규칙, 검증/발췌/anti-stale 3원칙, refs 발췌·블록 설계 협업 루프 포함)

## 경로 규칙 (개발자용)

- 번들 스크립트는 `${CLAUDE_PLUGIN_ROOT}/scripts/...` 로 참조한다(플러그인 위치 자동 해석).
- 글로벌 연구노트는 `~/.claude/연구_라이브러리/...` (사용자 홈) 에 누적한다.
- 절대경로(`C:\Users\...`)는 쓰지 않는다.

## 참고

- 이 플러그인은 **연구 도구 전용**이다. 위험명령 확인(guard-ask)·완료 알림음 같은
  범용 훅은 포함하지 않는다(개인 `~/.claude/settings.json`에서 따로 설정).
- 모든 커맨드/에이전트는 **추측 금지**(근거 없는 값은 `확인 필요`) 원칙으로 동작한다.
