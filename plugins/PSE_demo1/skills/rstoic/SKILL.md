# RStoic — 화학량론 전환율 반응기 (Aspen Plus)

> 이 스킬의 목적: **RStoic 블록을 이미 알고 있는 상태**가 되어, 사용자가 이 반응기가 무엇인지·어떻게 동작하는지 매번 설명하지 않게 한다. (반응기 입력값을 대신 채워주는 도구가 아니다.)

## 정의
- **화학량론(전환율) 반응기.** 반응 **속도론(kinetics) 없이**, 사용자가 지정한 **전환율(fractional conversion)** 만큼 반응물이 생성물로 바뀐다.
- 반응 평형도 풀지 않는다 — 오로지 "이 반응물의 몇 %가 이 화학량론대로 전환된다"만 적용한다.

## Specifications 탭 (운전조건)
설정할 수 있는 항목:

| 항목 | 비고 |
|---|---|
| Temperature (온도) | |
| Pressure (압력) | |
| Duty | 열부하 |
| Vapor fraction | 기상분율 |

- ⚠️ 위 4개는 **설정 가능한 항목 목록**이다. 실제로는 이 중 **2개 스펙을 지정**한다(예: 압력 + 온도, 또는 압력 + Duty, 또는 압력 + Vapor fraction). 나머지는 계산으로 결정된다.

**Valid phases (유효상)** — 다음 9가지 중 선택:
1. Vapor-only
2. Liquid-only
3. Solid-only
4. Vapor-Liquid
5. Vapor-Liquid-Liquid
6. Liquid-free water
7. Vapor-Liquid-free water
8. Liquid-Dirty water
9. Vapor-Liquid-Dirty water

## Reactions 탭 (반응 정의)
- 반응을 **양론계수(stoichiometric coefficient)를 포함한 반응식**으로 정의한다.
- 각 반응식 = "어떤 반응물이 **얼마나(전환율)** 전환되어 **어떤 생성물**을 만드는지"를 화학량론으로 구현.
- 전환율은 보통 **기준 반응물(key component) 기준 fractional conversion**으로 입력한다.

### `Reactions occur in series` 체크박스 — 직렬 vs 병렬
| 상태 | 동작 |
|---|---|
| ☑ **체크 (직렬, series)** | 반응식이 **순서대로** 진행. 앞 반응의 **생성물(출구 조성)이 다음 반응의 입력**이 된다. |
| ☐ **미체크 (병렬, parallel)** | 각 반응이 **원래 공급물 기준으로 독립** 전환. 같은 반응물을 여러 반응이 공유하면 전환율 합이 그대로 적용된다. |

**직렬일 때 계산 메모** — 같은 반응물에 대해 전환율 c₁, c₂, … 가 순차 적용되면 각 단계 직전 잔여량에 곱해진다:
- R3 직전 잔여 = (1−c₁)(1−c₂)…
- 예) c₁=0.04, c₂=0.0125 이면 R3 직전 잔여 = 0.96 × 0.9875 = 0.9480 → R3 전환 0.949367 적용 시 최종 = 0.9480 × 0.949367 = **0.9000 (90%)**.
- (병렬이면 이런 누적 없이 각 전환율이 원공급물에 그대로 적용된다.)

## 적용 시 기억할 것
- RStoic은 **수지(전환)만** 맞춘다 — 속도·평형·반응기 크기/체류시간은 모사하지 않는다(필요하면 RPlug/RCSTR/REquil).
- 직렬/병렬 선택이 **다중 반응이 같은 반응물을 공유할 때 결과를 바꾼다** — 반드시 의도와 일치하는지 확인.
- 사용자가 "RStoic" 또는 위 설정/동작을 말하면, 이 지식을 전제로 바로 대화한다(반응기 개념 재설명 불요).
